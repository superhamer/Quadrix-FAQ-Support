# Quadrix — Frequently Asked Questions

## General

**What is Quadrix?**
Quadrix is a task prioritization app based on the Eisenhower Matrix. It organizes your tasks into four quadrants: Urgent & Important, Important but Not Urgent, Urgent but Not Important, and Not Urgent & Not Important — helping you focus on what truly matters.

**Who made Quadrix?**
Quadrix was developed by Maher Ladsak, CMA, CPA(PH) in collaboration with Rork.com. All rights reserved.

**What platforms does Quadrix support?**
Quadrix is available on iOS and Android.

---

## Projects

**What is a project?**
A project is a collection of tasks organized across four quadrants. You can create multiple projects to separate different areas of your life or work (e.g., Personal, Work, Health).

**How do I create a new project?**
Tap the **+** button and enter a project name, or open the Project Panel via the ellipsis menu or by tapping the main header.

**How do I switch between projects?**
Swipe left or right on the main screen, or tap the page bar indicator at the bottom to jump to a specific project. You can also open the Project Panel to browse and sort all projects.

**How do I rename a project?**
Open the Project Panel and long-press the project name to edit it.

**How do I delete a project?**
Long-press the main trash icon to enter project delete mode.

**What is the homepage?**
When you have more than one project, a homepage is automatically generated. It provides a read-only summary of all tasks across all your projects.

**Is there a limit to how many projects I can create?**
There is no hard limit enforced by the app.

## Tasks

**How do I add a task?**
Tap the **+** button inside any quadrant. Type your task, then tap outside the field or tap the save icon to save it.

**How do I edit a task?**
Tap a task to open it for editing.

**How do I complete a task?**
Tap the checkbox next to a task to mark it as completed.

**How do I delete a task?**
Enter delete mode by tapping the trash icon, then select the tasks you want to remove. Deleted tasks are moved to the Recycle Bin.

**How do I move a task to a different quadrant or project?**
Long-press a task or use multi-select mode, then choose the destination quadrant or project.

**Can I reorder tasks?**
Yes. Enable the grip icon in the quadrant header, then drag the grip handle next to any task to reorder it.

**Can I reorder projects?**
Yes. Open the Project Panel and drag the grip handle to reorder projects.

**Is there an Undo feature?**
Single-level undo is available when tasks are moved to the Trash from a single quadrant, or when a task is transferred between quadrants.

## Quadrants

**What are the four quadrants?**
- **Urgent & Important** — Do these now.
- **Important, Not Urgent** — Schedule these.
- **Urgent, Not Important** — Delegate these.
- **Not Urgent & Not Important** — Eliminate or defer these.

**How do I expand or collapse a quadrant?**
Tap the quadrant header or swipe up/down on a quadrant to expand or collapse it.

**How do I minimize a quadrant?**
Swipe left or right within a quadrant to minimize it.

---

## Reminders & Calendar

**How do I set a reminder for a task?**
Tap the clock icon while adding or editing a task, then pick a date and time.

**Where can I see all my reminders?**
The Calendar view displays all tasks that have active notifications scheduled.

**Do notifications work in the background?**
Yes, push notifications are delivered even when the app is not open, subject to your device's notification permissions.

---

## Attachments & Voice Notes

**What types of attachments can I add to a task?**
You can attach images, files, and audio recordings. Each attachment can be up to **10 MB**.

**How do I attach a file or image?**
Tap the paperclip icon while adding or editing a task, then select from your photo library or files.

**How do I record a voice note?**
Tap the microphone icon while editing a task. You can record, play back, scrub through the timeline, and download voice notes.

---

## Search

**How do I search for a task or project?**
Tap the search icon to open the Search modal. You can search across all tasks and projects.

## Display & Accessibility

**Can I change the font size?**
Yes. In **Settings → Large Font Size**, toggle on to increase task text size by 25% across both minimized and maximized quadrants.

**Can I show or hide priority numbers?**
Yes. In **Settings → Priority Numbers Display**, cycle through: show on projects & tasks, projects only, tasks only, or hidden.

**Can I hide the grip icons?**
Yes. In **Settings → Grip Icons**, toggle the setting. When off, grip icons are only available via a manual toggle on each quadrant header.

**Can I reposition the + button?**
Yes. Long-press the + button to move it to the left, center, or right of the screen.

## Backup & Restore

**How do I back up my data?**

There are several options:
- **Download Current Project** — Exports only the currently active project.
- **Create System Backup for All Projects** — Creates a local backup of all projects. You can then save this file to a cloud service (iCloud, Google Drive, etc.) manually.
- **Auto Backup** — Enables automatic local backups on a Daily, Weekly, Monthly, or Quarterly schedule.

**How do I restore from a backup?**
- **Restore from System Backup** — Select a previously created local backup restore point.
- **Select and Upload Backup** — Pick a backup file from your device to restore from.

> ⚠️ Restoring a backup will override all existing tasks. This action is irreversible.

**How many cloud backups can I keep?**
Up to **3 cloud backups**. Each backup is available for a maximum of **6 months** and is automatically deleted when a new one is created.

**Will my backups be deleted if I uninstall the app?**
Yes. Deleting the app removes all cloud backups stored in the system. Always save a copy to an accessible external location (e.g., iCloud Drive, Google Drive, email) before uninstalling.

**Are auto backups safe against phone resets?**
No. Auto backups are stored in system storage and may be lost during a phone reset or app reinstall. Always create a manual backup and save it to an external location.

## Print

**Can I print my tasks?**
Yes. Use the print feature to preview and print a single project or multiple projects at once.

## Recycle Bin

**Where do deleted tasks go?**
Deleted tasks are moved to the Recycle Bin (Trash) and can be reviewed under the trash screen.

**How do I permanently delete all trashed tasks?**
Go to **Settings → Empty Recycle Bin**. This permanently deletes all tasks in the bin across all projects and cannot be undone.
